/gamemode

This command allows you to switch between the 3 gamemodes.

/enchant

The command will add an enchantment to the item the user is currently holding.

/give

This command allows a player to gift another player with the defined amount of the specified item.

/god

This command toggles 'god' status. In god mode you are invulnerable to all damage type

/time

Allows you to see and change the time on the server, either for a specific world or for all worlds.

/broadcast

Displays a specified message to all players on the server.

/vanish

When you type this command, become invisible ingame, and also hidden from most ingame commands.

/weather

This command allows the manipulation of the weather on your current world.

/banip

Bans the IP of a specified player.

/fly

When you type this command, your client will be informed it is allowed to use creative mode fly.

/seen

This command returns the time when the player last logged off.

/ban

Bans a specified player. Accepts partial names.

/unban

This command unbans the specified player.

/jails

This command returns a list of all current jail locations.

/socialspy

This command allows you to toggle the visability of other players direct messages and mail.

/tempban

This command allows you to ban a player for a specified amount of time.

/mute

Mutes or un-mutes a specified player. The time argument is optional, and will override any previous set time.

/jail

Sends a specified player to a specified jail. The time argument is optional, and will override any previous set time.

/tp

If the player has not disallowed teleportation this command will teleport you directly to a player.

/tphere

If the player has not disallowed teleportation this command will teleport you the player directly to you.

/co help

Display a list of commands in-game.

/co inspect

Enable the inspector. Type the command again to disable it. You can also use just "/co i".

/co rollback

u:<user> t:<time> r:<radius> a:<action> b:<blocks> e:<exclude>

Nearly all of the parameters are optional. Shortcut: "/co rb".

/co restore

u:<user> t:<time> r:<radius> a:<action> b:<blocks> e:<exclude>

Restoring can be used to undo rollbacks.

/co lookup

u:<user> t:<time> r:<radius> a:<action> b:<blocks> e:<exclude>

Lookups can be used to search through block data.

/kick

This command will kick the specified player off the server.

/afk

This command toggles your AFK status.

/help

The commands shown will be limited by permissions for essentials commands, and other plugins which provide permissions information.

/helpop

This command allows players to send a message to players with the matching helpop permission.

/list

This command lists the current player list, and gives the server slot/load count.

/msg

The message will be sent directly to the player and not displayed to others unless they have the socialspy permission.

/spawn

Teleports you to the location of your group's spawn.

/home

Teleports you to the location of where you last used the /sethome command.

/sethome

This command allows you to create a 'home' at your current location.
